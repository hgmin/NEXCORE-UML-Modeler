NEXCORE UML Modeler 사용자 가이드

1. 살펴보기

NEXCORE UML Modeler는 UML 2.1을 지원하는 유용한 그래픽 모델링 도구입니다. NEXCORE MDA Designer, NEXCORE MDA Develper와 연동하여 분석/설계 작업을 자동화하며, NEXCORE Framework 컴포넌트를 빠르게 생성할 수 있습니다. 비즈니스 로직을 구현하기 위해 모델 기반의 아키텍처와 자동화된 기능을 활용하여 생산성을 증대시킬 수 있습니다.
