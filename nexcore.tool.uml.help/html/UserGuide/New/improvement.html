NEXCORE UML Modeler 변경사항

2.1.0 - 2013/01/18

개선
다이어그램 편집기 같은 파일 내부의 다이어그램이 여러 개 열려있을 때, 한 다이어그램의 정보가 삭제되면 다른 다이어그램 편집기까지 모두 닫혔던 문제를 개선하였습니다.

2.0.1 - 2012/12/12

추가
공통 협업 시 모델 충돌이나 오류를 대비하여 자동으로 작업 공간 내의 UML 프로젝트가 백업되는 기능을 추가하였습니다. UML 모델러 구동 후 2분 뒤부터 10분 간격으로 백업을 하며, 모델러를 켜놓더라도 사용하지 않을 경우에는 10분이 지나면 백업하지 않습니다. 백업을 했더라도 72시간이 지난 백업 파일은 각 시간대별 최근 파일 1개만 남기고 삭제 처리됩니다.
개선
다이어그램 일반 다이어그램 에디터에서 UML 요소를 삭제하고 Undo하면 해당 요소가 실재하지 않는 것처럼 표시되는 문제를 개선하였습니다.
VOPC 다이어그램 VOPC 다이어그램 생성 시 미생성된 내부 UML 요소 때문에 해당 다이어그램을 편집기로 열 때 모델이 변경된 상태로 표시되는 문제를 개선하였습니다.
유스케이스 다이어그램 유스케이스 다이어그램 편집기를 열었을 때, 수정하지 않았어도 수정된 것처럼 표시되는 문제를 개선하였습니다.
시퀀스 다이어그램 1. 시퀀스 다이어그램에서 메시지 작성 시, 오퍼레이션 선택 다이얼로그에 오퍼레이션을 이름순으로 정렬하도록 개선하였습니다.
2. 프로젝트 탐색기에서 시퀀스 다이어그램으로 UML요소를 끌어다 놓을 때, 라이프라인의 순서가 맞지 않던 문제를 개선하였습니다.
3. 시퀀스 다이어그램의 복사가 가능하도록 개선하였습니다. 시퀀스 다이어그램 상위의 Collaboration 이상의 단위를 선택했을 때만 복사가 가능합니다.
특성 창 1. 매개변수 추가 시, 표의 빈 영역을 더블 클릭하여 행을 추가할 수 있도록 개선하였습니다.
2. 매개변수에 설정한 유형이 삭제가 되지 않는 문제를 개선하였습니다.
3. 여러 건의 매개변수를 동일한 유형을 설정하려고 할 때, 한 번에 설정이 가능하도록 개선하였습니다.
4. 리턴 방향인 매개변수의 유형 설정 시, 유형을 변경하면 매개변수의 이름도 함께 변경되는 문제를 개선하였습니다.

1.2.0 - 2011/04/08

추가
다이어그램 공통 1. 다이어그램에서 노드의 복사/잘라내기 & 붙여넣기 기능이 추가되었습니다.
2. 노드의 컴파트먼트(하위 구조) 사이즈 조절 기능이 추가되었습니다.
액티비티 다이어그램 액티비티 다이어그램이 추가되었습니다.
컴포넌트 다이어그램 컴포넌트 다이어그램이 추가되었습니다.
특성 창 다이어그램 내에서 선택하는 전체 UML 요소에 대해 모양 탭이 추가되었습니다. 노드의 색상 및 글꼴 색을 변경할 수 있습니다.
Outline 뷰 다이어그램 공통으로 사용되는 Outline 뷰가 추가되었습니다.
환경설정 액티비티/컴포넌트 다이어그램 환경설정이 추가되었습니다.
개선
다이어그램 일반 1. 연결선이 가로, 세로 직선이 아닌 대각선일 경우 끊김 현상이 나타나던 문제를 개선하였습니다.
2. 타원 형태의 노드일 경우 테두리에 끊김 현상이 나타나던 문제를 개선하였습니다.
시퀀스 다이어그램 1. Reply 메시지 이동 시 메시지가 사라지던 오류를 수정하였습니다.
2. Reply 메시지 생성 시 BehaviorExecutionSpecification이 생성되던 오류를 수정하였습니다.
3. Synch 메시지 생성 시 fragment가 반대로 생성되던 오류를 수정하였습니다.
4. Behavior 노드에서 메시지 생성 시 피드백이 나타나지 않던 오류를 수정하였습니다.
5. 메시지 생성 시 화면 밖으로 이동이 편리하게 개선되었습니다.
프로젝트 탐색기 1. 탐색기에서 다른 노드와 연결관계가 있는 노드를 Drag&Drop할 때 발생하던 오류를 수정하였습니다.
2. Type(클래스, 인터페이스)에서 함수 및 속성이 추가될 때 프로젝트 탐색기에서 Refresh가 되지 않던 오류를 수정하였습니다.
